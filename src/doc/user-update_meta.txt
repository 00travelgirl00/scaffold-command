wp-user-update_meta(1) -- Update a meta field for a WordPress user.
====

## SYNOPSIS

`wp user update_meta` <ID> <meta_key> <meta_value>

## OPTIONS

* `<ID>`:

	The ID of the user to update.

* meta_key:

	The meta_key to be updated
	
* meta_value:
	
	The new desired value of the meta_key

## EXAMPLES

	wp user update_meta 123 description "Mary is a WordPress developer"
